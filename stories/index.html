Stories Need a Local Server

This folder contains Storybook source files, not a directly viewable static page. To browse stories exactly as they are served by the website, run npm run serve:site and open http://localhost:8080/stories/.

If you already built Storybook with npm run build-storybook, you can open the static output from ../storybook-static/ using an HTTP server.

# Queens Strategy Snippets

Use this page to explain one tactical idea at a time instead of requiring users to play a full game state.

## 1. Crowns Block Touching Cells

A crown blocks all eight neighboring cells. This is often the first elimination wave to apply before row/column reasoning.

## 2. Dot vs Crown Meaning

- Dot: provisional rejection, useful to keep track of eliminations.
- Crown: committed placement that updates row, column, region, and touching constraints.

  <Story of={StrategyStories.DotVsCrownCycle} />

## Teaching Notes

- Introduce one rule per snippet.
- Keep captions short and concrete.
- Pair each snippet with one sentence that describes the decision the player should make.
